# Tools Store Admin Dashboard

Panel de control profesional para la gestión integral de la tienda de herramientas. Permite administrar el catálogo, controlar el inventario, ver reportes de ventas y configurar el sistema.

## 🚀 Tecnologías
- **React 19 & Vite**: Frontend ultrarrápido.
- **TypeScript**: Desarrollo seguro.
- **Tailwind CSS**: Estilizado moderno y responsivo.
- **Recharts**: Visualización de datos y estadísticas.
- **Lucide React**: Set de iconos profesionales.
- **Zustand**: Gestión de estado global.

## 🛠️ Variable de Entorno (Vercel)
Debes configurar esta variable en Vercel para que el panel pueda hablar con la API:

```env
VITE_API_URL=https://tu-api-prod.vercel.app/api/v1
```

## ✨ Funcionalidades
- **Dashboard**: Estadísticas reales de ventas y categorías.
- **Catálogo**: Gestión de productos con carga masiva (Excel).
- **Inventario**: Control de stock y facturas de compra.
- **Configuración**: Edición de datos bancarios, notificaciones y más.

## 📦 Instalación
1. `npm install`
2. `npm run dev`
